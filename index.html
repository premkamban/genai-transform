RAG Concepts

  • Embeddings: float vectors encoding semantic meaning
  • Cosine similarity: measures meaning, not keyword overlap
  • Chunking: fixed (512 chars), overlap (128), semantic split
  • Dense vs sparse retrieval: embeddings vs BM25 keyword
  • HyDE: embed a hypothetical answer, not the raw question
  • MultiQueryRetriever: 3 rephrasings → 3 retrievals → merge
  • RAGAS: faithfulness, answer_relevancy, context_precision

Rate Limit Handling

OpenAI returns 429 when you hit the rate limit. Without try/except your entire pipeline crashes. With it, you retry after 60 seconds.

Agent Tool Failures

A web search tool might fail on a bad URL. Agents need to catch the error, log it, and try a different search query instead of stopping.

JSON Parse Errors

LLMs sometimes return malformed JSON even in JSON mode. Catch json.JSONDecodeError and retry with a stricter prompt.

Production-Grade API Call
import time, json from openai import RateLimitError, APIError def safe_call(prompt, retries=3): for attempt in range(retries): try: r = client.chat.completions.create( model="gpt-4o-mini", messages=[{"role":"user", "content":prompt}], response_format={"type":"json_object"} ) return json.loads(r.choices[0].message.content) except RateLimitError: time.sleep(2 ** attempt) # exponential backoff except json.JSONDecodeError: prompt += "\nRespond ONLY with valid JSON." return None # failed after retries
